# /e2e - E2Eテスト実行

**e2e-runner** エージェントを起動して、Playwright MCPでE2Eテストを生成・実行する。

## このコマンドでできること

1. **テストジャーニー生成** — ユーザーフローのPlaywrightテストを生成
2. **E2Eテスト実行** — ブラウザでテストを実行
3. **アーティファクト取得** — スクリーンショット、動画、トレース
4. **フレーキーテスト特定** — 不安定なテストを特定・隔離

## このプロジェクトの主要テストシナリオ

1. **初期表示**: 「AI チャット」ヘッダー、空チャット画面、入力欄
2. **チャット送信**: メッセージ入力 → 送信 → ストリーミング応答
3. **設定パネル**: ⚙️クリック → モデル切替 → ダークモード切替
4. **画像生成ボタン**: 🖼️クリック → 準備中メッセージ

## ローカル開発サーバー

- URL: `http://localhost:8788`
- API: `GET /api/models`, `POST /api/chat`, `POST /api/image`
- 起動コマンド: `pnpm preview`（`wrangler pages dev dist`）

## 使用例

```
/e2e チャット送信から応答受信までのフローをテストして
```
